# Mini Projects

## Brief Description
This folder contains several small, real-world HTML projects designed to reinforce concepts learned in the HTML laboratory. Each project focuses on a specific aspect of web page development.

## Projects Included:
1.  **Contact Us Page (`contact_us.html`):** A form-based page for users to send messages.
2.  **Student Registration Page (`student_registration.html`):** A comprehensive form for student enrollment.
3.  **Admin Data Table (`admin_data_table.html`):** A table displaying administrative user data with basic styling.
4.  **FAQ Page (`faq_page.html`):** A page featuring a list of frequently asked questions using `details` and `summary` elements.

## Steps to Run
To view any of these mini-projects, simply open the respective `.html` file in your web browser.
