# Capstone Project: Personal Website

## Brief Description
This is a complete personal website built entirely with HTML. It integrates various HTML5 features and best practices for semantic structure, accessibility, and SEO. The website includes sections for personal introduction, skills, projects, experience/education (data table), and a contact form.

## Features:
*   **Semantic Layout:** Utilizes `header`, `nav`, `main`, `section`, `article`, `aside`, and `footer` elements.
*   **Images & Media:** Includes placeholder images and demonstrates how media could be integrated.
*   **Registration/Contact Form:** A functional contact form for visitors to get in touch.
*   **Data Table:** Presents structured information in an accessible HTML table.
*   **Accessibility & SEO-friendly:** Incorporates meta tags, `alt` attributes for images, and proper heading hierarchy.

## Steps to Run
To view your personal website, open `index.html` located in this folder in your web browser.
