Task 2.1: Navigation Bar with Flexbox

Learn how Flexbox creates flexible, responsive navigation layouts

Understanding Flexbox for Navigation

The navigation bar above uses Flexbox to align elements horizontally. Flexbox is perfect for creating one-dimensional layouts (rows or columns).

Notice how the navbar layout is responsive and adapts to different screen sizes. Each section (logo, menu, search, buttons) is positioned using Flexbox properties.

Navigation Structure:

<nav> <div class="nav-container"> <div class="logo">MyApp</div> <ul class="menu"> <li><a href="#">Home</a></li> ... more links ... </ul> <div class="nav-right"> <input class="search-bar" /> <div class="user-menu"> <button>Login</button> <button>Sign Up</button> </div> </div> </div> </nav>

CSS Flexbox Properties:

.nav-container { display: flex; justify-content: space-between; /* Space items evenly */ align-items: center; /* Vertically center items */ gap: 20px; /* Space between items */ } .menu { display: flex; gap: 5px; flex: 1; /* Take available space */ justify-content: center; /* Center menu items */ } .user-menu { display: flex; gap: 15px; align-items: center; }

Key Flexbox Concepts

display: flex - Turns a container into a flex container

justify-content - Aligns items along the main axis (horizontal for flex-direction: row)

align-items - Aligns items along the cross axis (vertical for flex-direction: row)

gap - Creates space between flex items

flex: 1 - Makes an item grow to fill available space

Real-World Applications:

  • Navigation bars (like our example above)
  • Header layouts with logo, menu, and buttons
  • Button groups and control panels
  • Card layouts and content grids
  • Form layouts with labels and inputs
  • Footer layouts with multiple sections

The navigation bar above is designed to be responsive. Try making your browser window narrower to see how Flexbox helps maintain a clean layout at different screen sizes.
