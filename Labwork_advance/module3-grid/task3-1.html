Understanding CSS Grid for Dashboard Layouts

CSS Grid is a two-dimensional layout system that's perfect for creating complex, structured layouts like dashboards. Unlike Flexbox which is one-dimensional, Grid can control both rows and columns simultaneously.

This dashboard uses a simple 2-column, 3-row grid structure:

Header (spans full width)
Sidebar
(250px)
Main Content
(1fr)
Footer (spans full width)

Grid Template Setup

The main grid container has two columns: a 250px sidebar and a flexible 1fr main area. The rows are defined as: auto (header), 1fr (content), and auto (footer).

.dashboard { display: grid; grid-template-columns: 250px 1fr; grid-template-rows: auto 1fr auto; gap: 0; min-height: 100vh; } header { grid-column: 1 / -1; /* Spans all columns */ grid-row: 1; } sidebar { grid-column: 1; grid-row: 2; } main { grid-column: 2; grid-row: 2; } footer { grid-column: 1 / -1; /* Spans all columns */ grid-row: 3; }

Grid Properties Explained

grid-template-columns: Defines the width of each column. 250px for sidebar, 1fr for flexible main area.

grid-template-rows: Defines the height of each row. auto, 1fr, and auto make header/footer size to content, main area takes remaining space.

grid-column / grid-row: Position items in the grid. Using "1 / -1" makes an item span all columns.

fr unit: "Flexible fraction" - takes up remaining space proportionally. 1fr means take 1 share of remaining space.

Real-World Dashboard Applications

This layout pattern is used in:

  • Admin Panels: Sidebar navigation with main content area
  • Project Management Tools: Multiple panes and sections
  • Analytics Dashboards: Header, sidebar, and content grid
  • Email Clients: Folder sidebar with email list and preview
  • IDE Editors: File explorer, code editor, and console
  • CMS Platforms: Navigation, settings, and content area
