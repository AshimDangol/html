Understanding Grid Spanning

Grid spanning allows items to take up multiple grid cells, either horizontally (grid-column) or vertically (grid-row). This is powerful for creating layouts where different sections need different amounts of space.

Spanning Properties:

/* Span 2 columns */ .card-wide { grid-column: span 2; } /* Span 2 rows */ .card-tall { grid-row: span 2; } /* Span 2 columns and 2 rows */ .card-featured { grid-column: span 2; grid-row: span 2; } /* Specific positioning */ .card { grid-column: 1 / 3; /* From column 1 to column 3 */ grid-row: 1 / 4; /* From row 1 to row 4 */ }

grid-column: span 2

Makes an item take up 2 columns of space horizontally.

.wide { grid-column: span 2; }

grid-row: span 2

Makes an item take up 2 rows of space vertically.

.tall { grid-row: span 2; }

Both Directions

Combine both properties to create large featured cards.

.featured {
grid-column: span 2;
grid-row: span 2;
}

Explicit Positioning

Use start and end positions for precise control.

.card {
grid-column: 1 / 4;
grid-row: 2 / 5;
}

Responsive Spanning

Change spanning in media queries for mobile layouts.

@media (max-width: 768px) {
.wide { grid-column: span 1; }
}

Auto Placement

Grid automatically fills gaps created by spanning items.

/* Items wrap to next row automatically */
