Understanding Grid for Statistics Dashboards

The statistics cards above use CSS Grid with grid-template-columns: repeat(4, 1fr) to create a flexible 4-column layout. This approach is perfect for responsive dashboards because the grid automatically adjusts the number of columns on different screen sizes.

CSS Grid Implementation:

.stats-grid { display: grid; grid-template-columns: repeat(4, 1fr); gap: 20px; } /* Tablet: 2 columns */ @media (max-width: 1024px) { .stats-grid { grid-template-columns: repeat(2, 1fr); } } /* Mobile: 1 column */ @media (max-width: 640px) { .stats-grid { grid-template-columns: 1fr; } }

Key Grid Concepts:

repeat() function: Creates multiple columns with the same size. repeat(4, 1fr) creates 4 equal-width columns.

1fr unit: "Flexible fraction" - represents an equal share of available space. With repeat(4, 1fr), each column gets 1/4 of the container width.

gap property: Creates consistent spacing between grid items. This is much cleaner than margin-based spacing.

Responsive grid: By changing the grid-template-columns in media queries, the cards automatically reorganize from 4 columns ‚Üí 2 columns ‚Üí 1 column on smaller screens.

üí° Auto-Responsive

Grid automatically wraps items when columns change in media queries. No need for complex flex-wrap logic.

‚èπÔ∏è Equal Sizing

Using 1fr ensures all cards are exactly the same width. Much simpler than calculating percentages.

üéØ Predictable Layout

CSS Grid is explicit about dimensions. You know exactly how many cards will fit on each row.

üîÑ Easy Modifications

Changing from 4 columns to 3, or 3 to 2, requires only a single CSS line change.

üìè Consistent Gaps

The gap property applies to all grid items at once, ensuring uniform spacing throughout.

üåê Browser Support

CSS Grid has excellent support in all modern browsers. Perfect for production applications.

Comparing Grid vs Flexbox for Cards:

/* With Flexbox: Need flex-wrap and complex calculations */ .cards { display: flex; flex-wrap: wrap; gap: 20px; } .card { flex: 0 1 calc(25% - 15px); /* Calculation needed */ min-width: 200px; } /* With Grid: Simple and clean */ .cards { display: grid; grid-template-columns: repeat(4, 1fr); gap: 20px; }

Real-world applications: Analytics dashboards, admin panels, KPI displays, metrics cards, performance monitoring, business intelligence platforms
